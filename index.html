What it does

Core Features

Everything in the terminal. No browser, no distractions.

Stack Overflow Search
Searches Stack Overflow using the official API and returns the top 8 most relevant questions, sorted by vote count and relevance.
requests
Top Answers Display
Fetches the top 3 answers for any selected question, ordered by votes. Shows accepted status, author name, and a 400-character preview of the answer body.
stackexchange api
QR Code Generator
Generates a PNG QR code from the Stack Overflow thread URL and saves it locally. Scan it on mobile to open the full thread instantly.
qrcode
Readability Analysis
Runs Flesch Reading Ease and Kincaid Grade Level analysis on the top answer. Shows word count, sentence count, and an Easy / Medium / Hard difficulty rating.
textstat
Paginated Results
Didn't find what you needed on the first page? Press [n] to load the next page of results, or [s] to start a completely new search without restarting the tool.
pagination
.env Configuration
Stores the Stack Exchange API key securely in a .env file using python-dotenv. Runs without a key too, falling back to 300 free daily requests automatically.
python-dotenv

Quick Start

Installation

6 steps, ready in 5 minutes.

01
Clone the Repository
git clone https://github.com/mashruf99/CLI_StackFlow_Digest.git
cd CLI_StackFlow_Digest
02
Create Virtual Environment
python -m venv venv
# Windows: venv\Scripts\activate
# Mac/Linux: source venv/bin/activate
03
Install Dependencies
pip install -r requirements.txt
04
Add API Key to .env
# .env
APP_NAME=CLI StackFlow Digest
VERSION=1.0.0
STACKOVERFLOW_KEY=your_key_here
05
Run the Application
python news_digest.py
06
